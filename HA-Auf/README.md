# HA-Auf

Neues Projekt mit eigenem Workspace.

## Struktur

- **HA-Auf.xcworkspace** – Xcode-Workspace (diesen in Xcode öffnen: **File → Open** → `HA-Auf.xcworkspace`)
- **HA-Auf.xcodeproj** – Xcode-Projekt
- **HA-Auf/** – Quellcode der iOS-App (SwiftUI)

## Öffnen in Cursor

Um in Cursor in diesem Projekt zu arbeiten:

1. **File → Open Folder…** (oder **Cmd+O**)
2. Ordner **HA-Auf** auswählen (der Ordner, in dem diese README liegt)
3. Bestätigen

Dann ist „HA-Auf“ dein aktiver Workspace.

## Öffnen in Xcode

Doppelklick auf **HA-Auf.xcworkspace** oder in Xcode **File → Open** → `HA-Auf.xcworkspace` wählen.

## App

Minimale SwiftUI-iOS-App mit einem Bildschirm „HA-Auf / Neues Projekt“. iOS 17+, Swift 5.
